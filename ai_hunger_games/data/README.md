# Data Directory

This directory contains simulation logs and results.

Files generated during simulation:
- `simulation_YYYYMMDD_HHMMSS.json` - Complete simulation data in JSON format
- `simulation_YYYYMMDD_HHMMSS.csv` - Round-by-round summary in CSV format
- `agents_YYYYMMDD_HHMMSS.csv` - Agent statistics in CSV format

These files are automatically created when you run the simulation.
